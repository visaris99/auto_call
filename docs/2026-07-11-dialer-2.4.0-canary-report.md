## 패키지 구성 확인

Windows publish 원본과 최종 Setup의 입력 폴더를 대조했다.

```text
adb/adb.exe          8,485,016 bytes
adb/AdbWinApi.dll      108,184 bytes
adb/AdbWinUsbApi.dll    73,368 bytes
```

- WSL과 Windows 빌드의 App/Core 소스는 줄바꿈을 제외하고 전부 일치했다.
- Windows 빌드 폴더와 WSL 최종 Setup의 SHA-256이 일치한다.
- `publish.bat`은 ADB 세 파일 중 하나라도 없으면 Setup 생성 전에 실패한다.
- `publish.bat`의 비대화식 `pause`를 제거하고 사용자 로컬 Inno Setup 경로를 지원한다.
- .NET 테스트 `111 passed`, Python 레거시 테스트 `43 passed`를 확인했다.
- Linux WPF Release 빌드는 경고 0, 오류 0으로 통과했다.
- Windows Release 테스트도 `111 passed`로 통과했다.

## 개발 PC 카나리 범위

Windows 사용자 `mirag`의 깨끗한 환경에서 이전 `b787d1bb...` 산출물을 설치해 아래 항목을 확인했다.

- 설치 경로: `%LOCALAPPDATA%\MilestoneDialer\app`
- 설치된 ProductVersion: `2.4.0`
- ADB 세 파일: 모두 존재
- 번들 ADB 실행: 정상, 현재 연결 단말 없음, 확인 후 daemon 종료
- 바탕화면 바로가기: 생성됨
- 제거 항목: `Milestone Dialer 버전 2.4.0`
- 앱 프로세스: 정상 실행
- 창 제목: `Milestone Dialer`
- updater 오류 로그: 생성되지 않음
- 검증 후 앱 프로세스: 정상 종료
- 사용자 kill switch: `TM_NO_AUTOUPDATE=1` 설정

최종 `7e294b...` 산출물은 같은 빌드 절차와 PE/ADB 검증을 통과했지만 다시 설치하거나 실행하지 않았다.
실제 전화 발신과 운영 고객 결과 저장은 사용자 결정에 따라 생략했다.

## 직원 PC 수동 배포

각 PC에서 다음 순서를 지킨다.

1. 진행 중인 통화가 없고 다이얼러가 종료됐는지 확인한다.
2. `setx TM_NO_AUTOUPDATE 1`을 실행하고 기존 다이얼러를 다시 종료한다.
3. Setup 파일의 SHA-256이 최종 값과 같은지 확인한다.
4. `milestone_dialer_setup_2.4.0.exe`를 실행한다.
5. 설치된 `milestone_dialer.exe`의 ProductVersion이 `2.4.0`인지 확인한다.
6. 로그인과 ADB 장치 상태까지만 확인한다.
7. PC 이름, Windows 사용자, 설치 시각, 로그인, ADB 상태를 배포대장에 기록한다.

해시 확인 PowerShell:

```powershell
(Get-FileHash .\milestone_dialer_setup_2.4.0.exe -Algorithm SHA256).Hash.ToLower()
```

모든 PC가 2.4.0으로 확인되기 전에는 다음 작업을 하지 않는다.

- CRM `latestVersion` 변경
- 서명 manifest 활성화
- `TM_NO_AUTOUPDATE` 제거
- 2.4.1 자동 배포

전환율 100% 확인 후 kill switch 제거:

```powershell
[Environment]::SetEnvironmentVariable("TM_NO_AUTOUPDATE", $null, "User")
```

## 남은 게이트

- 직원 PC별 설치 목록과 결과 기록
- 수동 배포 100% 확인
- 이후 2.4.1을 서명 manifest 기반 자동 업데이트 카나리로 검증

실단말 발신과 call attempt/call log 실측은 사용자 승인으로 이번 배포 게이트에서 제외했다. 따라서
ADB 제조사별 통화 상태 감지와 실제 발신 환경에 대한 잔여 위험은 월요일 운영 모니터링 대상으로 남는다.

`C:\milestone_build`는 직원 수동 배포와 최종 확인이 끝날 때까지 유지한다.
